# StratEdge Advisory | Strategic Consulting

A premium, Lond-inspired management consulting firm website for **StratEdge Advisory**. Built with modern aesthetics, smooth animations, and optimized for strategic excellence.

## 🚀 Live Demo
The website is automatically deployed via GitHub Pages:
**[https://ps1852.github.io/StratEdge-Advisory/](https://ps1852.github.io/StratEdge-Advisory/)**

## ✨ Features
- **Modern Corporate UI**: Deep navy and muted gold palette for a professional, high-end feel.
- **Service Pages**: Comprehensive descriptions of Corporate Strategy, M&A, and Operational Excellence.
- **Case Studies & Blog**: Strategic insights and success stories from our London partners.
- **Premium Animations**: Integrated Intersection Observer for smooth, scroll-triggered movements.
- **Fully Responsive**: Mobile-first design for seamless experience across all devices.
- **Auto-Deployment**: Integrated GitHub Actions workflow for continuous delivery.

## 🛠️ Technology Stack
- **Frontend**: HTML5, Vanilla CSS3, Javascript (ES6+)
- **Typography**: Inter (Body), Playfair Display (Headings) via Google Fonts
- **Deployment**: GitHub Pages & GitHub Actions

## 🏢 Company Profile
Based in Canary Wharf, London, StratEdge Advisory partners with ambitious leaders to solve their most complex challenges and capture their greatest opportunities.
